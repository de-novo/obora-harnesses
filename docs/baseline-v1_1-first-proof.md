# Baseline v1.1 — First Proof

## 한줄 결론

Baseline v1.1의 첫 proof point가 확보되었다.

현재 상태:
- **Baseline v1** = cross-repo replication established
- **Baseline v1.1** = first real proof obtained

v1.1의 의미는:
- docs-first low-risk lane 안에서
- **multi-paragraph bounded replace** 가 실제로 가능한지 검증하는 것

- `plant-companion`

### file
- `README.md`

### change type

## Why this matters

Baseline v1은 아래까지 검증했다.
- safe-create
- exact-replace
- small block replace
- paragraph-bounded replace

v1.1은 그 다음 단계다.
즉 한 문단을 넘는, 그러나 여전히 bounded된 문서 블록 수정이
실제 repo에서도 가능한지 보기 시작한 것이다.

이번 proof는 그 첫 사례다.

---

## Not yet established as baseline

아직 v1.1은 확립(established)된 baseline이 아니다.

이유:
- 현재는 단일 repo proof 1건뿐이다
- cross-repo replication이 아직 없다

즉 지금 상태는:
- **v1.1 candidate pattern** 은 보였다
- 하지만 **v1.1 replication baseline** 이라고 부르기엔 이르다

---

## Recommended next move

v1.1을 확립하려면 다음이 필요하다.

1. `polysona` 또는 `obora-todoapp`에서 multi-paragraph bounded replace 1건 추가
2. 가능하면 세 번째 repo까지 반복
3. wording refinement only / no semantic expansion 원칙 유지
