# Cross-Repo Proof Points

## Proven across real repos
- docs-first scope isolation
- hold-zone separation
- safe-create support docs
- single-line exact replace
- single-sentence exact replace
- small contiguous block replace
- paragraph-bounded replace
- re-read based verification

## Proven repos
- `polysona`
- `obora-todoapp`

## Current safe generalization
- low-risk documentation lane only
- single-file bounded changes only
- no semantic expansion baseline

## Not yet generalizable
- code/runtime modify
- workflow/config modify
- multi-file batching
- multi-paragraph or semantic-heavy rewrites

## Best next move
- replicate once more on a third real repo, or
- test one multi-paragraph bounded replace in docs-first scope
