# Ralph Loop 10회 반복 목표

## 목표 정의

10회 반복 내에 **작업 완료 + 검증 통과**를 달성하는 것.

## 성공 기준 (체크리스트)

각 반복마다 다음을 측정:

| # | 기준 | 측정 방법 | 통과 기준 |
|---|------|-----------|-----------|
| 1 | progress.md 읽기 | 파일 접근 로그 | 100% |
| 2 | progress.md 쓰기 | 파일 수정 로그 | 100% |
| 3 | guardrails.md 준수 | 위반 횟수 | 0회 |
| 4 | 작업 진척도 | 완료된 하위 작업 수 | 증가 추세 |
| 5 | 검증 통과 | validation.json | passed: true |

## 반복별 목표 단계화

```
반복 1: 작업 이해 및 계획 수립
반복 2-3: 핵심 구현 (MVP)
반복 4-6: 기능 확장 및 개선
반복 7-8: 버그 수정 및 안정화
반복 9: 최종 검증
반복 10: 완료 또는 실패 보고
```

## 실패 시 대응

| 실패 유형 | 대응 |
|-----------|------|
| 컨텍스트 오염 | 강제 교체 (토큰 60% 이상) |
| 동일 실패 반복 | guardrails.md에 규칙 추가 |
| 진척 없음 | 작업 분해 (더 작은 단위로) |
| 검증 실패 | 구체적인 수정 지시 추가 |

## 측정 지표

```json
{
  "iteration": 5,

}
```
