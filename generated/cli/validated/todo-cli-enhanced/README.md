# Todo CLI — Enhanced Workflow Target

This directory is reserved for the enhanced long-running workflow version of the Todo CLI.

## Intended workflow
- planner expands the product spec
- contractor writes task/acceptance contract
- generator implements the current scope
- evaluator checks CLI behavior and storage semantics
- repair loop revises failures

## Planned artifacts
- `spec.md`
- `task-contract.json`
- `qa-report.json`
- `comparison-summary.md`
- runnable CLI implementation files

## Purpose
This artifact will be compared against:
- `generated/cli/experimental/todo-cli-single-agent`

The goal is to measure whether an enhanced workflow improves completeness, usability, and robustness for a real CLI application.
